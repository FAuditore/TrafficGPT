# TrafficGPT

## Deployment

### Installation

TrafficGPT 无需安装，可以选择直接将代码 clone 到本地运行。

```Powershell
clone https://github.com/lijlansg/TrafficGPT.git
cd TrafficGPT
```

TrafficGPT 的运行需要如下的软件支持：

- Python
- SUMO
- PostgreSQL

同时，请安装所需第三方库：

```Powershell
pip install -r requirements
```
